# Fairway Rewards

The modern golf reward engine where every Stableford point counts toward legendary prizes and life-changing charity impact.

## Features
- Stableford Score Tracking
- Automated Reward Draws
- Charity Support Integration
- Stripe Subscriptions
- Premium Golf-Centric UI

## Setup
1. Setup your Supabase project using `database/schema.sql`.
2. Configure `server/.env` (see `server/.env.example`).
3. Install dependencies: `npm install` in both folders.
4. Run: `npm start` in server, `npm run dev` in client.
